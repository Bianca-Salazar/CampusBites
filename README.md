# VECI YA

## Descripción 
Proyecto colaborativo con el fin de ayudar a estudiantes a encontrar comida económica cerca de la universidad 

## Objetivo
Facilitar la búsqueda de restaurantes, cefeterías y promociones estudiantiles 

## Problema que resuelve
Muchos estudiantes tienen dificultades para encontrar lugares de comida que se adapten a su bolsillo y se encuentren cerca de la universidad, esto provoca pérdida de tiempo, mayores gastos y dificultad para encontrar con rapidez opciones dentro de su jornada académica.

## Público Objetivo
Estudiantes universitarios

##Funciones principales
- Buscar lugares de comida cercanos
- Mostrar precios accesibles
- Ver promociones
- Mostrar calificaciones
